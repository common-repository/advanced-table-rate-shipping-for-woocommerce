== Description ==

The WooCommerce table rate shipping plugin is an add-on to your WooCommerce store and an easy-to-use extension. Table rate shipping for Woocommerce, unlike other plugins, offers an ideal and easily understood settings panel that lets users calculate the shipping costs effortlessly.
Table rate shipping WooCommerce lets the customers experience advanced and flexible shipping features in the Cart and Product check-out pages. Set up various shipping rates based on the criteria such as weight, price, location, item count, or shipping class.

== How to manage the settings in the Woocommerce table rate shipping plugin? ==

To control the options and settings, the admin can navigate to WooCommerce > Advance Table Rate Shipping sub-menu, then add New Method Here, the admin can add new shipping zone or select shipping zone from pre listed shipping zone. Once done, the user can opt for the preferred shipping method based on multiple conditions like days, holidays, delivery, etc.

Installation of "WooCommerce Table Rate Shipping" can be done either by searching for "WooCommerce Table Rate Shipping" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Activate the plugin through the 'Plugins' menu in WordPress

= Does WooCommerce Table Rate Shipping offer compatibility with the WPML? =
Of course, the table rate plugin for WooCommerce offers due compatibility with the WPML plugin. Since the plugin is tested, it can be used to build multilingual sites without issues. 

= Is it possible to disable all the shipping methods in the Woocommerce cart page with your Table Rate shipping extension? =
Yes, you can use the WooCommerce shipping table plugin, to disable the shipping methods.  Set the ‘Disable other methods’ option to disable the Woocommerce shipping options on the cart page.
